# WP Backup Command

This WP-CLI package allows you to back up plugins or themes by creating a zip or tar.gz archive.

## Installation

`wp package install mwender/wp-backup-command`

## Usage

To back up a plugin:

`wp plugin backup <plugin-slug>`

To back up a theme:

`wp theme backup <theme-slug>`
